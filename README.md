# Live_iOS_Play_SDK-4.12.0
获得场景云直播SDK
## 在官方Demo的基础上改动内容如下：
- 去除pch文件内的所有引用，并将其引用添加到相应的头文件中。
- 视频加文档的直播形式改动，小窗播放改为底部交互区域展示文档或者视频，并兼容小窗模式。
- 移除关于全景视频VR相关的内容。
